# ADSL
Here are some instructions to run our code. Our code is based on PAT, thanks for their excellent work.
## 1. Clone this repo
git clone https://github.com/STUDY1231/ADSL.git
## 2. Prepare your environment
conda create -n adsl python==3.10

conda activate adsl

install the requirements
## 3. Download the datasets
Duke and Market1501
## 4.Quick Start
train with the train.py

test with the test.py
